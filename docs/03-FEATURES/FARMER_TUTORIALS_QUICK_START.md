# 📚 Farmer Tutorials - Quick Start Guide

## ✅ What's Implemented

A comprehensive tutorial system with **5 step-by-step guides**, **6 language support**, and **voice narration** to help farmers master the Soil2Crop app!

## 🎯 Features at a Glance

### **5 Core Tutorials:**

1. **📤 How to Upload Soil Report** (3 min)
   - Upload PDF/image reports
   - Review extracted values
   - Generate recommendations

2. **🌱 Viewing Crop Recommendations** (4 min)
   - Understand crop suitability
   - Compare different crops
   - Check profit estimates

3. **📈 Checking Market Prices** (3 min)
   - Find best mandi prices
   - View price trends
   - Plan your sale

4. **📄 Viewing Government Schemes** (4 min)
   - Discover available schemes
   - Check eligibility
   - Apply online/offline

5. **🔔 Reading Alerts & Notifications** (2 min)
   - Understand alert types
   - Take appropriate action
   - Stay informed

## 🗣️ Voice Narration Demo

### **How It Works:**

1. Click "Listen" button
2. Browser voice activates
3. Reads tutorial title
4. Reads each step sequentially
5. Stops when complete or on "Stop Voice"

### **Voice Settings:**

- **Speed:** 0.9x (slightly slower for clarity)
- **Pitch:** 1.0 (normal)
- **Language:** Auto-detected from app language
- **Volume:** System volume

### **Example (Hindi):**

```
क्लिक करें: "सुनें"
आवाज़: "मिट्टी रिपोर्ट अपलोड कैसे करें। चरण 1: मिट्टी रिपोर्ट पर जाएं..."
```

## 📞 Troubleshooting

### **Common Issues:**

**Q: Voice not working?**  
→ Try Chrome/Firefox browser. Check volume. Safari has limited support.

**Q: Video not loading?**  
→ Check internet connection. May be region-blocked.

**Q: Wrong language?**  
→ Clear cache. Change language in Settings. Refresh page.

**Q: Can't expand cards?**  
→ Refresh page. Check JavaScript console for errors.
